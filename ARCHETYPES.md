# ARCHETYPES.md

Champions Reg M-A (VGC 2026 / ダブル / **テラスタル無し**) で軸になりうるアーキタイプ素材の棚卸し。Phase 0「方向お任せ」相談に入った時、ここから候補を提示する。

メタ採用率は時点情報のため `find_meta_threats` で都度引き直すこと。本ドキュメントは **どの素材がロスター内に存在するか** の構造的情報。

## 2. 場 (Terrain)

**ロスター内に Surge 系特性 0**。エレキ/グラス/ミスト/サイコフィールドは **全部技で 5T 張り**。場中心の構築は維持コスト高、ピンポイント運用 (例: サイコフィールドで優先技遮断) が現実的。

## 提案ガイドライン

- 軸候補は **3-4 アーキタイプを並列提示** → ユーザーに選んでもらう (スコア順禁止 / プリミティブの組み合わせ思想)
- どの軸でも Phase 1 (`get_pokemon` + `compute_type_matchup`) → Phase 2 (`find_pokemon` で補完) → Phase 3 (`get_pokemon_moves`) → Phase 4 (`damage_calc`) → Phase 5 (完成) のフロー
- メタ採用率は `find_meta_threats` で再取得 (本ドキュメントは時点スナップショット、2026-04 ベース)
- ロスター外提案禁止 — `find_pokemon` には常に `champions_only: true` または `tiers`
